# Algoritme - List

## Hva er en list(e)?

A list is an ordered or unordered data structure with elements separated by a comma and enclosed within square brackets

## Metoder:
### Obligatoriske:
- removeFirst
- removeLast
- remove
- first
- last
- contains
- isEmpty
- size
- iterator

### UnOrdered:
- addToFront
- addToRead
- addAfter

### Ordered:
- add
